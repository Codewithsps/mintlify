# Mintlify Documentation Website Clone

## Project Overview

This project is a **desktop-first documentation website clone** inspired by the Mintlify platform.  
The objective was to recreate the **structure, layout, and content hierarchy** of a modern documentation site with a strong focus on readability, UI consistency, and visual accuracy.

The implementation uses **pure HTML and CSS in a single file**, without any external frameworks or libraries.

## Sections Recreated

The following sections were implemented to closely match the original Mintlify website structure:

1. Top Navigation Bar  
   - Logo  
   - Navigation links  
   - Call-to-action buttons  

2. Hero Section  
   - Announcement badge  
   - Headline and description  
   - Email input with CTA  

3. Documentation Preview  
   - Sidebar navigation  
   - Main documentation content  
   - Card-based layout  

4. Trusted By Section  
   - Enterprise brand logos  

5. Features Section  
   - Alternating content and visuals  
   - AI-focused messaging  

6. AI Assistant Preview  
   - Mock chat interface  

7. Enterprise Section  
   - Security, compliance, and partnership highlights  

8. Customer Story Section  
   - Case study and statistics  

9. Industry Section  
   - Industry-agnostic positioning  

10. Final Call-To-Action  
    - Conversion-focused CTA  
    - Supporting features  

11. Footer  
    - Multi-column navigation  
    - Legal and company links
